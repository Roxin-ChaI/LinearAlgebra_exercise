# 线性代数重点练习版

这个文件夹是从原始 `线代练习题.tex` 中整理出来的复习重点版。

## 文件

- `线代重点练习题.tex`：母题版题单源码，题面保留英文，并标注原始章节、小节和题号。
- `线代重点练习题.pdf`：编译后可直接阅读的版本，若已生成。
- `original_problem_index.csv`：从原始 `线代练习题.tex` 生成的题号索引，方便核对出处。
- `pic/`：重点版中仍然需要的少量图片资源。
- 编译产生的 `.aux`、`.log`、`.out`、`.toc` 等文件也会保留在本文件夹中。

## 整理原则

原始题库覆盖面很广，适合完整训练；这个版本更适合复习时抓主干。当前版本按“母题”标准重新筛选：优先保留做透以后能迁移到同知识点其他题目的代表题。

题目正文默认不翻译成中文。每道题前面的 `Original:` 行会给出：

- Chapter
- Subsection
- Problem number

每道题前面的 `母题覆盖` 行会说明它覆盖哪一类题型，帮助你判断这道题为什么值得优先做。

保留题目时主要看四点：

1. 是否对应章节核心概念。
2. 是否能代表一类常见计算方法。
3. 是否能暴露容易混淆的概念。
4. 是否能串起后续章节，例如消元到秩、正交到最小二乘、特征值到 SVD。

## 标签说明

- `[Core]`：第一轮复习优先完成，当前共 58 题。
- `[Drill]`：掌握核心题后用于巩固，当前共 25 题。
- `[Challenge]`：用于检查理解深度，不适合赶进度时硬刷，当前共 9 题。

## 建议用法

先按章节完成所有 `[Core]`，再根据薄弱点补 `[Drill]`。
如果只是为了快速回忆整门课，优先做每章前半部分的 `[Core]` 题即可。
