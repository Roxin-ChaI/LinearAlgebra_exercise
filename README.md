# 📘 Serge Lang - A First Course in Calculus（习题整理）

## ✨ 项目初衷

在阅读 *Introduction to Linear Algebra* 的过程中，我发现页数众多且不便查找：
👉 课后习题分散、查找不方便，做题和回顾时体验不太友好。

于是我决定做一件简单但实用的事——
把书中的课后习题**重新整理并用 LaTeX 排版**，让它们更清晰、更易查阅，也更适合日常练习和检索。

这个项目本质上就是：
📌 **把“难用的题目列表”，变成“顺手的练习册”**
